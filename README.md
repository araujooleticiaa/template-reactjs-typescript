## 🏁 Para rodar o projeto:

Clone este repositório em sua máquina:

```bash
$ git clone https://github.com/araujooleticiaa/template-reactjs-typescript.git
```

cd `template-reactjs-typescript` e rode:

```bash
yarn
```

para iniciar:

```bash
yarn start
```
